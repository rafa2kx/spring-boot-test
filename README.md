# Getting Started
1. Create a new MySQL Database
2. Create a new application.properties file.
3. Set the following properties:
    1. DB_HOST for database address host. (Eg. localhost)
    2. DB_PORT for database port. (Eg. 3306)
    3. DB_NAME for the name of the newly created database(Eg. gateways)
    4. DB_USER (Eg. root)
    5. DB_PASSWORD (Eg. secret)
4. Run command `java -jar ./gateway-0.0.1.jar`
5. Enjoy!

* [API Documentation]. Deploy the app and go to /docs/index.html.
